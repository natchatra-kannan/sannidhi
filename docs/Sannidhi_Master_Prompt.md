# SANNIDHI - MASTER PROJECT PROMPT

## Project Overview

We are rebuilding an existing Glide-based web application called SANNIDHI as a professional Flutter Web application.

## Technology Stack

* Frontend: Flutter Web
* Backend: Supabase
* Version Control: GitHub
* Hosting: Free-friendly deployment (Cloudflare Pages or similar)
* Development Environment: Flutter SDK + Android Studio (tooling)

## Goals

* No Glide dependency
* No credit-based limitations
* Free-friendly architecture
* Scalable and maintainable
* Future support for Android and iOS from the same Flutter codebase

## Current Status

* Flutter SDK installed and working.
* Android setup nearly complete.
* Existing Glide application serves as the reference implementation.

## Development Approach

1. Design architecture before coding.
2. Create reusable components.
3. Build responsive web-first UI.
4. Use Supabase for authentication and data storage.
5. Implement role-based access where needed.
6. Maintain clean folder structure and scalable code.

## Authentication

* Email login flow
* Google login support
* Email PIN/OTP verification flow

## Requested Workflow

When continuing this project:

1. Review architecture.
2. Review database schema.
3. Review navigation tree.
4. Build page-by-page.
5. Generate production-quality Flutter code.
